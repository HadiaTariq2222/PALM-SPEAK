About Our Project:

The title of our project is PalmSpeak

Introduction:

In a world considered by fast developments in technology and communication, there is a need to bridge communication gaps. This gap commonly exists between deaf and mute people and in those who talk through verbal language. Communication is a fundamental human need. In Pakistan, like several other countries, the Deaf people face a trial which is unavailability to a regular and broadly known sign language system. Due to this unavailability of standard language, they face a lot of social problems, but this also restricts them in employment and empowerment.

We will empower the Deaf and Muted community with a PSL hand gesture recognition which will facilitate unified communication with hearing individuals. We will create a Deaf and Muted community with a deep learning-based model with a user-friendly interface available for both deaf and muted community and normal users. Our system will ensure the standard sign language for communication based on PSL hand gestures.

To represent hand gestures different types of methods such as vision based and sensor based are used but in this project we will use vision based gesture recognition system in which we will capture the image of the userâ€™s hand and then will adjust the size apply the deep learning based algorithm on that. vision based hand gesture recognition has made remarkable progress in the field of hand gesture recognition.

Image processing technique will apply on it for extracting features of hands and then recognize the gestures.The input video that is captured from camera is transformed to a frame. Set of predefined features are extracted from each of these frames. Data pre-processing operation is then applied for extracting the features from it and cleaning, these extracted features is then stored for training. Extracted features are altered from different gesture. Input video isthen given to the hand gesture recognition system which convert the video into frames and extract out all the possible feature from the frames. These features are then compared and with the already stored features in the database to interpret the hand gesture
